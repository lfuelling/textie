textie
======

Ein kleines Textadventure von @[lfuelling](https://github.com/lfuelling) und @[YuleZ](https://github.com/YuleZ).
Ziel des Spiels ist, das Rätsel zu lösen.

### Mögliche Befehle

- hilfe
- nimm [*gegenstand*]
- benutze [*gegenstand*]
- untersuche [*gegenstand* | raum | inventar]
- vernichte [*gegenstand*]
